# Ressources de Conception et Documentation Academic

Ce repertoire centralise les livrables de conception, les modelisations UML et les documents academiques liés au developpement de l'extension de gestion des conges.

## 1. Valeur de la Documentation (Aspect Humain)

La presence de ces ressources temoigne du travail de reflexion et de modelisation en amont du codage. Elles permettent :
- Une transmission fluide de la logique metier aux nouveaux developpeurs.
- Une justification des choix d'architecture aupres des parties prenantes.
- Un support d'apprentissage et de soutenance pour l'evolution du projet.

## 2. Contenu du Repertoires

### 2.1 Modelisation UML (Diagrammes)
Le dossier diagramme-uml et le fichier modelisation-uml.mdj contiennent les specifications visuelles de l'application :
- Diagrammes de Classes : Structure des entites (Utilisateur, Demande, Historique).
- Diagrammes de Séquence : Flux de validation d'une demande de conge.
- Diagrammes de Cas d'Utilisation : Interactions entre l'employe, le manager et l'administrateur.

### 2.2 Rapport de Soutenance
Le fichier RAPPORT_DE_STAGE_FEUKOUO_FEUSSI_RYAN_NEUVILLE_BTS2_2026_GL_AKWA.docx constitue la documentation textuelle globale du projet sous un angle academic et professionnel, detaillant le contexte, les defis techniques rencontrés et les solutions apportées.

## 3. Integration dans le Cycle de Vie
Ces documents ne sont pas statiques ; ils doivent etre mis a jour lors de chaque evolution majeure du modele de donnees ou du workflow de validation pour rester synchrones avec l'implementation reelle.
